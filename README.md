# Call Center Intelligence

A **world-class Call Center Intelligence Control Tower** that unifies cases from multiple channels into a single system, then surfaces **what's happening now** via a social-media style feed, trending topics, smart alerts, semantic search, and one-click sharing/escalation to management.

## Overview

This project is a fully functional prototype designed to demonstrate what a "world-class call center" experience looks like - moving beyond traditional BI dashboards to a **Live Intelligence Feed** that feels like X/Twitter for customer service operations.

## Key Features

- **Live Feed** - Social-media style feed with alerts, trending topics, and highlights
- **Real-time Feel** - Polling/SSE for live updates even with mock data
- **Semantic Search** - Natural language search across cases
- **Chat Assistant** - "Ask the Control Tower" insight assistant
- **Smart Alerts** - Spike detection, threshold alerts, urgency flagging
- **Trending & Prediction** - Topic trend analysis with forecasting
- **Sharing & Escalation** - Management visibility workflows

## User Roles

1. **PM/PO (Admin View)** - Full access to all BUs, channels, and configuration
2. **BU Manager** - BU-scoped view with escalation capabilities
3. **Call Center Supervisor** - Operational view for incoming spikes and queue health

## Tech Stack

- Next.js (Option A - Prototype)
- SQLite with mock/seed data
- Server-Sent Events for real-time updates
